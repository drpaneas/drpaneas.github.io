VIM plugin for writing in Markdown

In the early beginning, I found writing posts in markdown format quite tricky. Things can become quite helpful by using an editor that provides you with features such as syntax highlighting and HTML monitoring. Apparently, there are noumerous plugins out there, but for me, ‘vim-instant-markdown’ seems to do the job, just great.

Install

In order to install it through Vundle, edit your .vimrc and put the git source of the plugin:

 " plugin for MarkDown
Plugin 'suan/vim-instant-markdown'

Force .md files to be treated as .markdown

By default, only the .markdown extension is typically recognized as markdown filetype. Given the fact that I’m one of those people who prefer to use the .md extension as an alternative; VIM also needs to be condifured, so to treat the content of my posts like any other .markdown file.

That has been said, edit your .vimrc file and add this:

" Detect *.md as *.markdown
autocmd BufNewFile,BufReadPost *.md set filetype=markdown

To test if it works, edit (or create) a file with .md extension and observe if the syntax has been highlighted or not. Also, your web-browser should immediately pop-up and start generating – on-the-fly – a webpage, based on the markdown syntax of the file.
