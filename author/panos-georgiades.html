Βασικό configuration του Nginx

Date Δευ 02 Φεβρουαρίου 2015 Category Nginx Tags nginx / server

Σε αυτό το άρθρο θα μιλήσουμε για βασικές ρυθμήσεις που πρέπει να κάνουμε στον Nginx. Οι ρυθμήσεις αυτές έχουν περισσότερο την έννοια του optimization, παρά του configuration, καθώς ο Nginx έρχεται pretty much configured, οπότε εδώ θα μιλήσουμε τις βέλτιστες ρυθμήσεις. Για να κάνω μία εισαγωγή, θα μιλήσουμε για πράγματα όπως: πόση μνήμη έχουμε διαθέσιμη, πόσους πυρήνες (cpu) θα διαθέσουμε, και πώς αυτά θα επιδράσουν στον συνολικό αριθμό των processes, threads, κ.α. πραγμάτων που σχετίζονται με τους πόρους και την λειτουργία του Nginx στο σύστημά μας. Τονίζω το "σύστημά μας" γιατί ο σκοπός μου δεν είναι να σας δώσω ...

Εισαγωγή στον Nginx

Date Πεμ 22 Ιανουαρίου 2015 Category Nginx Tags nginx / server

Σε αυτό το άρθρο θα μιλήσουμε για τον Nginx1 HTTP Webserver (ω ναι, υπάρχουν κι άλλοι webservers πέρα από τον Apache). Όταν λέω ότι θα κάνουμε μία εισαγωγή, εννοώ ότι θα ξεκινήσουμε την εξερεύνηση απαντώντας σε κάποια από τα βασικά ερωτήματα, όπως:

  • Σε τι διαφέρει από τον Apache;
  • Ποια είναι να χαρακτηριστικά του;
  • Γιατί να ασχοληθούμε μαζί του;

Καταρχάς, ο Nginx δεν είναι καινούριος. Υπάρχει εδώ και αρκετό καιρό στην πιάτσα, συγκεκριμένα το dev ξεκίνησε το 2002 και το 2004 κυκλοφόρισε η πρώτη του έκδοση. Ωστόσο το τελευταίο διάστημα βλέπω ότι έχει αρχίζει να τραβάει πάνω του αρκετή προσοχή ...
